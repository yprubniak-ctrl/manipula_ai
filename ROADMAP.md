# Manipula Platform - Product Roadmap

## Vision

To become the smallest self-sufficient digital product unit — a compact, orchestrated AI system that can independently build and ship software at startup speed.

## 📊 Metrics & KPIs

### Product Metrics
- **Project Success Rate**: % of projects that complete successfully
- **Time to Deployment**: Average time from idea to live URL
- **User Satisfaction**: NPS score
- **Cost per Project**: Average LLM cost
- **Agent Performance**: Success rate by agent type

### Business Metrics
- **Monthly Active Users**: Active users per month
- **Projects Created**: Total projects created
- **Revenue**: MRR and ARR
- **Churn Rate**: User retention
- **Enterprise Adoption**: Enterprise customers

### Technical Metrics
- **API Response Time**: p95 latency
- **System Uptime**: 99.9% target
- **Agent Execution Time**: Average duration
- **Error Rate**: % of failed executions
- **Cost Efficiency**: $ per successful project
